# L01 — Real-World Applications of Computer Vision

## Problem Statement

Explore how computer vision is applied in a major real-world domain, demonstrating understanding of how CV systems perceive and process visual information in practical environments.

## Approach

Written essay analyzing self-driving autonomous vehicles as a primary case study for computer vision. The paper covers how cameras and deep learning models work together with complementary sensors (LiDAR, radar, GPS) through sensor fusion to enable reliable perception.

Key areas addressed:
- How convolutional neural networks interpret road environments
- Sensor fusion combining visual and non-visual inputs
- Benefits: reduced human error, consistent traffic law adherence
- Challenges: adverse weather, ethical decision-making, privacy concerns from continuous data collection

## Key Findings

Computer vision is the core perception layer in autonomous vehicles — without it, the system cannot identify lane markings, pedestrians, signs, or other vehicles. However, CV alone is insufficient; it requires fusion with radar and LiDAR to remain reliable across varying lighting and weather conditions. Ethical and regulatory frameworks are as important as the technical solutions.

## Technologies / Concepts

- Convolutional Neural Networks (CNNs)
- Sensor fusion (cameras + LiDAR + radar + GPS)
- Object recognition and scene understanding
- Autonomous systems
